The shown code and data belongs to the Final Project for the Visual Analytics Project

------------------------------------------------------------------------------------------------
The folder web_scraping contains the python code for web scraping hotel data from Google Maps 
	as well as the four output files.

The folder additional_data contains all additional datasets that are added to the scraping data 
	during the data cleaning process.

The folder data_cleaning contains the two Alteryx files used for the data cleaning as well as their final output.

The folder dashboard contains the final tableau dashboard as well as the two data files needed.
